Introduction

In order to make simple websites on the internet, you need to learn the basics of HyperText Markup Language (HTML). In this document, you will learn what HTML is and how to make a basic website. It will also go over what CSS is and how to style your website.

The learning path is HyperText Markup Language or HTML -> Cascading Style Sheets CSS.
What you should know
  • General understanding of the internet
  • How to type
  • How to do google searches
  • Some understanding and skill in VS Code

These basic skills will help you achieve the level of programming you want. Learning how to learn is one of the most important skills.

All programmers will be constantly learning new libraries, lines of codes, algorithms, etc. This is just part of being a programmer.
HTML Introduction

HyperText Markup Language (HTML) is a language used to structure websites. Think of HTML as the foundation of the website. It is one of the many layers of a website.

Elements

Element is everything from the start of a tag until the end tag.

For example: html, body, head, div, title, meta, p, etc.
Create your first website

In order to begin coding, you must first have VS Code (click on this link) downloaded. Once downloaded, you can begin by writing your first line of code!

Trail and error is how you will learn. Understanding how to use VS Code and debugging is essiential to grow.
CSS Introduction

Cascading Style Sheets (CSS) is used to style and layout websites. This is your second layer which adds your unique styling to the website. Be creative.

Style your website

In order to link your css file to your html file, you must first use the 'link' element. This is necessary because it allows the computer to link both files in order to style your website.

Linking is extremely important for knowing what you have changed to your website. When linking use 'href', be sure to copy the file name letter for letter.
